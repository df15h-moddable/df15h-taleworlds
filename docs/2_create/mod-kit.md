
The toolkit for Mount & Blade II Bannerlord is installable from the "Tools" menu in the Steam Library.

Bannerlord has C# and XML APIs.

The devkit includes these tools [2] ...

- Scene editor
- Mesh editor
- Material editor
- Model/Animation viewer
- Skeleton editor
- Replay editor
- Particle editor
- Atmosphere Editor
- Cloth editor
- Path editor
- Resource Browser
